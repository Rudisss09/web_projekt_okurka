04

O Projektu

📊

Přesná data

Pravděpodobnosti vychází z oficálně potvrzených dat Valve pro CS2 bedny (Container Odds). Šance jsou zákonem povinně zveřejněné pro hráče z EU.

🎓

Školní projekt

Web byl vytvořen jako ročníková práce z předmětu Webové technologie. Postaven na čistém HTML5, CSS3 a Vanilla JavaScript bez frameworků.

⚠️

Zodpovědná hra

Bedny jsou hazard s velmi nízkými šancemi na hodnotné předměty. Tento nástroj slouží k vzdělávání o pravděpodobnostech, ne k podpoře hraní.

Použité technologie

  • HTML5 Sémantické značky, ARIA, strukturovaná data
  • CSS3 Flexbox, Grid, animace, CSS proměnné, Media Queries
  • JS ES6+ Vanilla JS, moduly, Canvas API, Intersection Observer
  • SEO Meta tagy, Open Graph, Twitter Cards, JSON-LD
  • A11y WCAG 2.1 AA, ARIA, klávesnicová navigace
